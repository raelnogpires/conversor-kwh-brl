## Repository Map

Um mapa completo do repositório está disponível em `codemap.md`, na raiz do
projeto.

Antes de trabalhar em uma tarefa, leia `codemap.md` para entender:

- a arquitetura e os pontos de entrada do projeto;
- a responsabilidade de cada diretório;
- o fluxo de dados entre as interfaces e as regras de domínio;
- os pontos de integração entre os pacotes.

Para trabalhar em uma pasta específica, leia também o `codemap.md` existente
dentro dela.
